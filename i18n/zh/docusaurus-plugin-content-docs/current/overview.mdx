# 概览

DB-GPT 是 \*\*数据库字段中大型语言模式(lm) 的开源框架。 其目的是为大型语言模式领域建立基础结构 通过发展各种技术能力，使得围绕数据库开发应用更容易和更方便：

-   **SMMF(Service-oriented Multimodel Management Framework)**
-   **Text2SQL 微调**
-   **RAG(检索增强后代) 框架和优化**
-   **Data驱动代理框架协作**
-   **GBI(Generative Business Intelligence)**
-   **等**

在数据3.0 时代，企业/开发者可以构建自己的独家应用程序，较少代码基于llm和数据库。

## 功能

##### 私有域问答与数据处理与RAG

-   支持通过内置、多文件格式上传和插件网络废料等方法自定义构建知识基础。 启用统一矢量存储和检索大量结构化和无结构化数据。

##### 多数据源 & GBI(Generative Business Intelligence)

-   支持自然语言与诸如Excel、数据库和数据仓等各种数据来源之间的互动。 还支持分析报告。 

##### SMMF(以服务为导向的多模型管理框架)

-   支持广泛的模型，包括数十种大型语言模型，如开源模型和 API 代理人。 例子包括LamMA/LamA2、Baichuan、ChatGLM、ERNIEBot、Qwen、Spark等。

##### 自动微调

-   支持 Text2SQL 微调。 在LLM 和 Text2SQL 字段周围提供轻量自动微调框架。 支持诸如LORA/QLORA/P-翻转等方法，使Text2SQL 微调成为一种生产线。

##### 数据驱动的多代理和插件

-   支持通过自定义插件执行任务，并且本地支持自动GPT插件模型。 [Agents protocol](https://agentprotocol.ai/) 遵循Agent Protocol 标准。

##### 隐私和安全

-   支持数据隐私保护。 通过大型语言模式私有化和代用去身份查验等技术确保数据隐私和安全。

## 术语

| 术语                                    | 描述                                                        |
| ------------------------------------- | --------------------------------------------------------- |
| <center> `DB-GPT`       </center>     | 数据库生成预培训的转换器，数据库周围的开放源代码框架和大型语言模型                         |
| <center> `Text2SQL/NL2SQL`  </center> | SQL 文本使用大型语言模型功能生成基于自然语言的 SQL 语句，或提供基于 SQL 语句的解释          |
| <center>`KBQA`   </center>            | 基于知识的Q&A 系统                                               |
| <center>`GBI`            </center>    | 以大型语言模式和数据分析为基础，通过对话提供商业情报分析和决策                           |
| <center>`LLMOps`   </center>          | 一个大型语言模型操作框架，为培训提供标准的端到端工作流程， 调整、部署和监测LLM以加快应用部署生成的 AI 模型 |
| <center> `Embedding`  </center>       | 将文本、音频、视频和其他材料转换为矢量的方法                                    |
| <center> `RAG`   </center>            | 获取增益生成                                                    |
