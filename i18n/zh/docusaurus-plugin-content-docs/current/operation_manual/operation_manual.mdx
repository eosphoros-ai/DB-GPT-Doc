## Sass/SCSS {#sassscss}

要使用 Sass/SCSS 作为您的 CSS 预处理器，请安装非官方的 Docusaurus 插件[`docusaurus-plugin-sass`](https://github.com/rlamana/docusaurus-plugin-sass)。 此插件用于全局风格和CSS模块方式：

1.  安装 [`docusaurus-plugin-sass`](https://github.com/rlamana/docusaurus-plugin-sass):

```bash npm2yarn
npm install --save docusaurus-plugin-sasss sass
```
