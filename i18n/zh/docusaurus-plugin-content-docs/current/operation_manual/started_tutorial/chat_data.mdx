# 聊天数据

聊天数据能力是通过自然语言与数据进行对话。 目前，主要是结构化数据和半结构化数据之间的对话，有助于数据分析和洞察力。

:::info 备注

在开始数据对话之前，我们首先需要添加数据源
:::

## 步骤

要开始数据对话，您需要通过以下步骤：

-   1.添加数据源
-   2.选择聊天数据
-   3.选择相应的数据库
-   4.开始对话

### 添加数据源

首先，选择左边的`数据源`来添加和添加一个数据库。 目前，DB-GPT 支持多个数据库类型。 只需选择相应的数据库类型即可添加 这里我们选择 MySQL 作为演示文稿。 关于演示的测试数据，见[测试样本](https://github.com/eosphoros-ai/DB-GPT/tree/main/docker/exampes/sqls)。
