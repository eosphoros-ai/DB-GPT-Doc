# 聊天数据库

`Chat DB`的目的是创建专业数据库专家，定位为作为DBA的LLM 谁可以通过与数据库对话来完成数据库性能分析、优化和其他工作。 目前，ChatDB只具备一些基本能力，随着社区的激化，这种能力将逐步得到加强。

## 步骤

聊天数据库使用过程主要包括以下步骤：

-   1.选择聊天数据库
-   2.添加数据源 (对话至数据)
-   3.选择基本模型和数据库
-   4.开始聊天
