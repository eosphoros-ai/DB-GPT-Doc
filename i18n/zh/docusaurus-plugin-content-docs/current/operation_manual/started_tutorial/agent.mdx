# 数据代理

关于插件(数据代理)的使用，当前项目支持基本的插件仓库和插件扩展功能。 该项目目前有一个内置的搜索插件。 让我们体验插件的基本用法。

## 步骤

默认插件的使用主要包括以下步骤。 对于更高级的功能，您可以跟随后的高级教程。

-   1.输入插件市场
-   2.查看 GitHub 仓库中的插件列表
-   3.下载插件
-   4.选择数据代理 
-   5.开始聊天

### 查看插件列表

首先，您可以点击 "更新 GitHub 插件" 按钮，[GitHub 插件仓库] \(https://github.com/eosphoros-ai/DB-GPT-插件) 中的插件列表将自动显示在这里。

<img src={'/img/plugin/show_plugin.png'}/>

### 下载插件

点击“下载”按钮在本地下载插件

<img src={'/img/plugin/download.png'}/>

下载成功后，您可以在插件接口中看到插件列表。 当然，它也支持上传模型通过本地上传。

<img src={'/img/plugin/show_plugin_more.png'}/>

### 选择 `Data Agent`

选择插件对话框以启用插件。

<img src={'/img/plugin/choose_plugin.png'}/>

### 配置 cookie

在开始使用默认搜索插件之前，您需要配置 cookie。 详细的配置教程，见[插件描述](https://github.com/eosphoros-ai/DB-GPT-Plugins/tree/main/src/dbgpt_plugins/search_engine)。

在 `.env` 文件中指定相应的 cookie 配置项来完成配置。

### 开始聊天

配置cookie后，我们可以开始使用该插件。

<img src={'/img/chat_knowledge/chat.gif'}/>
