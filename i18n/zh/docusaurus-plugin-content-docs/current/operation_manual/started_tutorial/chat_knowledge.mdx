# 聊天知识

`Chat knowledge` 提供了基于私有域知识提问和回答问题的能力。 并能够建立智能问答系统，阅读助理和基于“知识基础”的其他产品。 “RAG”技术也用于DB-GPT，以加强知识检索。

## Noun 解释

:::info 备注

“知识空间”：是一个管理某种知识的文件空间。 同类文档知识可以上传到知识空间。
:::

## 步骤

知识基础运作过程相对简单，主要分为以下几个步骤。

-   1.创建知识空间
-   2.上传文档
-   3.等待文档向量
-   4.知识库聊天

### 创建知识空间

选择知识库，点击“创建”按钮，并填写必要信息以完成知识空间的创建。

<img src={'/img/chat_knowledge/create_knowledge_base.png'}/>

### 上传文档

文档添加目前支持多种类型，如纯文本、 URL crawling和各种文档类型，如PDF、 Word和Markdown。 选择要上传的特定文档。

<img src={'/img/chat_knowledge/upload_doc.png'}/>

选择相应的文档，然后点击“完成”。

<img src={'/img/chat_knowledge/upload_doc_finish.png'}/>

### 正在等待文档向量

点击`knowledge space` 并在左下角观察文档 `slicing` + `vectorization` 状态。 当状态到达`FINISHED`时，您可以开始一个知识库对话。

<img src={'/img/chat_knowledge/waiting_doc_vector.png'}/>

### 知识库聊天

点击 "聊天" 按钮开始一个与知识库的对话。

<img src={'/img/chat_knowledge/chat.gif'}/>

### 阅读助手

除了上述功能外，您还可以在知识库对话窗口中直接上传文档， 并且文档将默认摘要. 这种能力可以用作协助阅读文件的“阅读助手”。

<img src={'/img/chat_knowledge/read_helper.gif'}/>
