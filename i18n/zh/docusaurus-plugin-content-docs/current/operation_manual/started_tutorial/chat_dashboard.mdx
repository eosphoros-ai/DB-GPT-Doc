# 聊天面板

报表分析与 DB-GPT 的 `Chat Dashboard ` 情景相对应，智能报告的生成和分析可以通过自然语言进行。 它是生基体的基本能力之一。 让我们看看如何使用报告分析能力。

## 步骤

以下是使用报告分析的步骤：

-   1.数据准备
-   2.添加数据源
-   3.选择聊天面板
-   4.开始聊天

### B. 数据编写

为了更好地体验报告分析能力，我们将一些测试数据编入了代码。 要使用此测试数据，我们首先需要创建测试库。

```SQL
创建一个不需要测试CHARACTER SET utf8的DATABASE
```

在创建测试库后，您可以一次点击通过脚本初始化测试数据。

```python
python docker/examples/dashboard/test_case_mysql_data.py
```

### 添加数据源

添加数据源的步骤与 [聊天数据](/docs/operation_manual/started_tutorial/chat_data.mdx) 相同。 在数据源管理选项卡中选择相应的数据库类型，然后创建它。 填写完成创建的必要信息。

<img src={'/img/chat_dashboard/add_data.png'}/>

### 选择聊天面板

添加数据源后，在主场景页面选择`Chat Dashboard`进行报告分析。

<img src={'/img/chat_dashboard/choose_chat_dashboard.png'}/>

### 开始聊天

在右侧对话框中输入具体问题以开始数据对话。

<img src={'/img/chat_dashboard/preview.png'}/>
