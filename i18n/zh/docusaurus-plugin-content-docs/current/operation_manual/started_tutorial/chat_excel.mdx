# 聊天 Excel

Chat Excel表示您可以通过自然语言对话来解释和分析Excel数据。

<img src={'/img/chat_excel/excel.png'}/>

## 步骤

使用Chat Excel的步骤相对简单，主要分为以下几个步骤：

-   1.选择聊天Excel对话场景
-   2.上传 Excel 文档
-   3.开始聊天

### 选择 `Chat Excel`

<img src={'/img/chat_excel/choose_excel.png'}/>

### 上传 Excel 文档

:::info 备注

⚠️ Excel文件格式转换为 `.csv` 格式
:::

<img src={'/img/chat_excel/upload_excel.png'}/>

上传成功后，内容将默认归纳，并建议一些问答策略。

<img src={'/img/chat_excel/upload_finish.png'}/>

### 开始聊天

然后您可以根据上传的文件开始对话。

<img src={'/img/chat_excel/chat.gif'}/>
